# Ascii Battleship Game
A ascii battleship game built in Java that can be played in the terminal.

## Usage:

```bash
java -jar Ascii-Battleship.jar
```

## Run on Windows:

Just run the exe!

## Install on Linux:

Aur: https://aur.archlinux.org/packages/ascii-battleship

```bash
yay -S ascii-battleship
```
