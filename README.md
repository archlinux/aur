# Build and Install Manually
```bash
$ git clone https://github.com/bgta/aur-gnome-shell-extension-laine-git.git
$ cd aur-gnome-shell-extension-laine-git
$ makepkg
$ sudo pacman -U gnome-shell-extension-laine-git-*-any.pkg.tar.xz
