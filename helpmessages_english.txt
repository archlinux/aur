* Output of the command 'balongflash -h':

== %< ==

The tool is designed for modem firmware chipset Balong V7

balongflash [options] <file name to download or directory with files>

 The valid keys are:

-p <tty> - serial port for communication with the loader (by default, /dev/ttyUSB0)
-n - multifaylovoy flashing mode of the specified directory
-g# - to set the digital signature (-gl - description of the parameters)
-m - display firmware file card and finish the job
-e - disassemble the firmware file into sections without headings
-s - disassemble the firmware file into sections with headings
-k - do not restart the modem firmware by the end of
-r - is forced to restart the modem firmware without partitions
-f - flash even if CRC errors in the source file
